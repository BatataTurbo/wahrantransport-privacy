Wahran Transport - Data Deletion Request

Users can request deletion of data related to Wahran Transport by contacting: transportalgerien@protonmail.com

How to request deletion

Please send an email including:

  • App name: Wahran Transport
  • Request type: data deletion
  • Optional context, such as a feedback or stop verification message you submitted

Data that can be deleted

  • Stop verification reports
  • Written feedback submitted through the app
  • Analytics events where deletion is technically possible

Data that may be retained

  • Anonymous aggregated statistics that cannot be linked to an individual user
  • Security logs required to protect the service, kept for a limited period

We normally process deletion requests within 30 days.
